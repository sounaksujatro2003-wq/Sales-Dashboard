Overview

The Retail Sales Dashboard 2024 is a Power BI project developed to analyze and visualize retail business performance. It provides insights into sales, profit, customer behavior, and regional trends, enabling informed and data-driven decision-making.

Objectives
Analyze overall sales and profit performance
Identify top-performing product categories and regions
Examine monthly and quarterly sales trends
Evaluate order conversion across different sales stages
Present geographical insights at state and city levels
Key Features
Key Performance Indicators (KPIs): Total Sales, Maximum Profit, Product Count
Category Analysis: Sales distribution across Electronics, Furniture, Clothing, and Food
Time Analysis: Monthly and quarterly performance trends
Regional Analysis: Comparative performance across North, East, West, and South regions
Sales Funnel: Visualization of order stages from prospect to closure
Geographical Analysis: Profit distribution across states and cities
Detailed Data Table: Transaction-level insights
Interactive Filters: Region and city-based slicers for dynamic analysis
Tools and Technologies
Power BI for data visualization and dashboard development
Microsoft Excel (or dataset source) for data storage and preprocessing
DAX (Data Analysis Expressions) for calculated measures and KPIs
Dataset Description

The dataset includes the following fields:

Order ID
Product Name
Category
Quantity
Sales Amount
Profit
Customer Name
Region and City
Order Status
Date (Month, Quarter, Year)
Key Insights
Electronics is the leading category in terms of revenue contribution
The East region generates the highest profit
The third quarter (Q3) shows peak profitability
The sales funnel indicates a strong order conversion rate
Usage Instructions
Open the .pbix file using Power BI Desktop
Use available filters (Region and City) to explore specific segments
Analyze KPIs and visualizations to derive insights
